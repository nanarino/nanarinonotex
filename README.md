# ☘ 自用初学笔记-Ex
 
自用学习笔记，基于 vitepress@1.0.0 构建，包含

- 飞镖🎯
- kotin🈁
- 长生不老药🍇



## 本地运行

安装依赖

```bash
pnpm i
```

脚手架命令

```bash
# serve with hot reload at localhost:8080
pnpm docs:dev

# build for production with minification
pnpm docs:build
```

可以将配置文件中的`base: '/notebook2/'`改为`base: '/'`
