# nanarinote-flutter
 
学习笔记，基于 vitepress@1.0.0-alpha.47 构建，包含

- dart
- Flutter


## 本地运行

安装依赖

```bash
pnpm i
```

启动命令（或运行 `docs_dev.cmd` ）：

```bash
# serve with hot reload at localhost:3000
pnpm docs:dev
```
